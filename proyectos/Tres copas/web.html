Tres copas

Crea tres copas:

  • A partir de un cilindro: añadiendo bucles y escalando
  • A partir de un círculo: extruyendo los vértices
  • Usando una curva de Bezier (o un path ) con screw

Comprueba la orientación de las normales y si hay problemas corrígelas Aplica el modificador de solidificar (solidify) a todos los objetos, si es necesario, para darles un grosor como si fueran de barro.
